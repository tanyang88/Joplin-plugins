# Joplin插件仓库

这是官方的Joplin插件库。每30分钟在整点和半小时更新一次。

## 安装

要安装这些插件，请打开桌面应用程序，然后转到Configuration屏幕中的“plugins”部分。然后，您可以搜索任何插件并从那里安装它。
